# NeoVim Settings

## Installation

Use `git clone` to copy into ~/.config/nvim for Linux or ~/AppData/Local/nvim for Windows.

Then in NeoVim run the `:PlugInstall` command to install all plugins.

### Plugin Requirements

The following tools are needed for the plugins to work properly:
* fzf
* ripgrep
* fd
* C compiler (for treesitter)
* a Nerd Font
